# Dotfiles

This is my neovim configuration.

## Usage

```bash
$ git clone git@github.com:nsr888/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
$ make prepare
$ make nvim-install
```

## Requirements

- Mac
- Git
- Make
- Nvim (version >= 0.5)
