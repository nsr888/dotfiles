# Dotfiles

This is my neovim configuration.

## Usage

```bash
$ git clone git@github.com:nsr888/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
$ make nvim-install
```

## Requirements

- Linux/Mac
- git, ripgrep, fd
- Nvim (version >= 0.9)
